Projects

CV Generator

I built the React cv / resume builder web app to practice using the useState hook in React. Users can create and manage their resumes and save as pdf when ready. There is a default view loaded when the app is first loaded to give users an idea of how the final result would turn out. One feature about React that stood out in this project for me was the need for every list item to have a unique id. I used the UUID library to achieve this behavior. I decided to separate the inputs and display for each section (personal details, work, education, and skills) into their own components to make the code more readable and maintainable. I also styled each component separately, although with vanilla CSS. I also included a localStorage helper function to persist user input in different sessions. I used Vite as my build tool and deployed the app on Vercel. Overall I thoroughly enjoyed building this web app and hope users enjoy using it too!

Memoji

Memoji is React memory game I designed and built to practice using the mainly the useEffect hook although I also used useState. Itâ€™s derived from two words memory and emoji. I used the axios library to get data from the emoji API. To win, users must click each of the 12 emojis at most once (the emojis reshuffle after each click). After each win, a new set of emojis are displayed and after 3 consecutive losses, new emojis are loaded. One of my main takeaways from this project was the use of a .env file for hiding API keys and integrating that with Vercel. This was a fun and gamified approach to learning and I hope users can create good memories on Memoji.

Todo List

I built the Todo List web app to practice vanilla JavaScript. Users can add new tasks, mark them as completed, delete tasks and filter them based on their state: all tasks, active tasks, or completed tasks. It features a drag and drop list and a dark mode toggle. The app also uses localStorage to persist user inputs between sessions. The app is hosted on GitHub pages. I enjoyed building this app and learnt a lot about vanilla JavaScript, particularly in this case how localStorage behaves when dealing with objects and how to implement drag and drop functionality.

Weather App

The Weather web app is an app I built to practice AJAX. Users can search for locations and get weather information for that day and a 4 day forecast. By default the weather information for Accra, Ghana is shown. The background image of the app changes based on the current weather conditions. It uses the Fetch API to make requests and consumes data from the Virtual Crossing Weather API. I chose that API because it gives a much longer forecast and additional weather data like humidity and visibility which I included in my app. The app features geolocation conversion, which means converting the entered location name into geographical coordinates using the OpenStreetMap API to give weather data for remote areas that may not be on the Virtual Crossing Weather API. Overall the weather app is a great app I hope users will enjoy using.
